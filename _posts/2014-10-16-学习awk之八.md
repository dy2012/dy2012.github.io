计数登录用户数：

{% highlight awk %}
~$ awk 'BEGIN { while("who" | getline) ++n; print n }'
{% endhighlight %}

获取当前运行时间记录在变量d中。
{% highlight awk %}
~$ awk 'BEGIN { "date" | getline d; print d }'
{% endhighlight %}

计数文件行数,其中filename是要计数的文件名，双引号不能省。

{% highlight awk %}
~$ awk 'BEGIN { while(getline <"filename" > 0) ++n; print n }'
{% endhighlight %}

忽略#include ...行，打印代码文件，gsub是处理日志文本的利器。

{% highlight awk %}
~$ cat program
{% endhighlight %}
{% highlight awk linenos %}
/^#include/ {
	gsub(/""/, "", $2)
	while (getline x <$2 > 0)
		print x
	next
}
{ print }
{% endhighlight %}

{% endhighlight awk %}
~$ awk -f program filename
{% endhighlight %}
 
之前介绍过awk支持命令行参数，还提供内建变量，ARGC，ARGV意义也和c类似，下面是打印命令行参数，其他玩法，可以参照c。

{%  highlight awk %}
BEGIN {
	for(i = 1; i < ARGC; i++)
		printf "%s, ", ARGV[i]
	printf "\n"
}
{% endhighlight %}
